PLACEHOLDER FOR APP ICONS

This directory should contain the following icon files for the PWA:

1. icon-192.png - 192x192 pixel app icon
2. icon-512.png - 512x512 pixel app icon

These icons should be square PNG images with the Zazaki game logo.
You can create these icons using any image editing software.

A simple design could be:
- Blue (#3498db) background
- White text or symbol representing language learning
